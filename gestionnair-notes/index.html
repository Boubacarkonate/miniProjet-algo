Gestionnaire de Notes

  • But : Créer une application simple pour ajouter, afficher et supprimer des notes.
  • Étapes :
    1. Permettre à l'utilisateur d'ajouter une note avec un titre et une description
    2. Afficher toutes les notes dans une liste.
    3. Ajouter un bouton pour supprimer une note spécifique.
  • Concepts pratiqués : Manipulation des objets et tableaux, DOM, événements.
