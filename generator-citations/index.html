Générateur de citation aléatoire

  • Description : Affichez une citation aléatoire d'une liste prédéfinie à chaque rechargement ou clic de bouton.
  • Points d’apprentissage : Tableaux, sélection aléatoire, manipulation du DOM.
  • Fonctionnalités supplémentaires : Utiliser une API pour récupérer des citations (ex. API de Quotes) pour des citations illimitées.
