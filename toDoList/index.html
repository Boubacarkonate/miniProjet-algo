Gestionnaire de liste de tâches (To-Do List)

  • Description : Créez une application de gestion de tâches où les utilisateurs peuvent ajouter, modifier, et supprimer des tâches.
  • Points clés d’apprentissage :
    • Manipulation des structures de données (tableaux pour stocker les tâches).
    • Gestion des événements et manipulation du DOM pour afficher les tâches.
    • Algorithmes simples pour organiser et filtrer les tâches (par ordre de priorité, de date).
  • Fonctionnalités supplémentaires : Ajouter des catégories de tâches, un calendrier, ou une fonctionnalité pour cocher les tâches terminées.
