Jeu de Pierre-Papier-Ciseaux

  • But : Créer un jeu où l'utilisateur joue contre l'ordinateur.
  • Étapes :
    • Demander à l'utilisateur de choisir entre "pierre", "papier" ou "ciseaux".
    • Générer un choix aléatoire pour l'ordinateur.
    • Comparer les choix pour déterminer le gagnant selon les règles du jeu.
  • Concepts pratiqués : Conditions, génération aléatoire, logique de comparaison.
