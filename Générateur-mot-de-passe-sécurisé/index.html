Générateur de mot de passe sécurisé

  • Description : Générez un mot de passe aléatoire avec différents critères (longueur, majuscules, chiffres, caractères spéciaux).
  • Points d’apprentissage : Boucles, conditions, chaînes de caractères.
  • Fonctionnalités supplémentaires : Copier le mot de passe dans le presse-papier, ajouter une barre pour la force du mot de passe.
