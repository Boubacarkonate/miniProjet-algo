Recherche dans un Tableau

  • But : Créer une fonction qui recherche un élément spécifique dans un tableau.
  • Étapes :
    • Définir un tableau de valeurs.
    • Demander à l'utilisateur un élément à rechercher.
    • Parcourir le tableau pour trouver cet élément et indiquer s’il est présent ou non.
  • Concepts pratiqués : Boucles, conditions, manipulation de tableaux.
